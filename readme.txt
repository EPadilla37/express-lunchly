Lunchly App
Lunchly is a basic restaurant reservation management app built using Node.js, Express, and PostgreSQL. 
It allows users to view and manage customer information, make reservations, and view reservation details.

Features
View a list of customers.
Add new customers with their details.
Edit existing customer information.
View customer details, including reservations.
Make new reservations for customers.

Getting Started

Clone the repository: git clone https://github.com/yourusername/lunchly-app.git

Navigate to the project directory: cd lunchly-app

Install dependencies: npm install
Set up the PostgreSQL database by running the provided SQL scripts.
Configure the database connection in the db.js file.
Start the server: npm start
Open your web browser and visit: http://localhost:3000

Usage
Browse to the homepage to view a list of customers.
Click on a customer to view their details and reservations.
Edit customer information or make new reservations using the provided forms.
Technologies Used
Node.js
Express.js
PostgreSQL
Nunjucks (template engine)
